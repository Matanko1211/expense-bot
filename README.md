# 💰 בוט ניהול הוצאות לוואצפ
### 💎 גרוש גרוש הופך לרכוש

בוט וואצפ חינמי לניהול הוצאות אישיות — עם מעקב תקציב, יעדים חודשיים, והתראות אוטומטיות.

---

## ✨ יכולות הבוט

| פיצ'ר | תיאור |
|-------|--------|
| ➕ הוסף הוצאה | שלח `85 סופר` — הבוט מזהה קטגוריה לבד |
| 📊 סיכום חודשי | כמה הוצאת לפי קטגוריה מול תקציב |
| 📅 סיכום יומי / שבועי | מה הוצאת היום או השבוע |
| 💼 מצב תקציב | Progress bar לכל קטגוריה |
| 🎯 יעדים חודשיים | הגדר כמה אתה מכוון להוציא |
| ⚠️ התראות | אלרט אוטומטי כשמגיעים ל-65% מהתקציב |
| 🗂️ 7 קטגוריות | אוכל, בריאות, קניות, דיור, תחבורה, בידור, דלק |
| ➕ קטגוריות מותאמות | הוסף קטגוריות משלך |
| 👥 מרובה משתמשים | כל משתמש מקבל נתונים נפרדים |
| 💾 שמירה אוטומטית | הנתונים נשמרים גם אחרי restart |

## 🛠️ טכנולוגיות

- **[Baileys](https://github.com/WhiskeySockets/Baileys)** — חיבור לוואצפ ללא Chrome
- **Node.js 20+**
- **Express** — שרת HTTP להצגת QR
- **שמירה לקובץ** — `data.json` נשמר אוטומטית
